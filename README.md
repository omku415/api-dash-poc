# API Dash – OpenAPI Template Generator (PoC)

## Overview

This project is a Proof of Concept (PoC) for automating API integration workflows.

It processes OpenAPI specifications and generates ready-to-use API request templates.

## Features

- OpenAPI parsing
- Endpoint extraction
- Data normalization
- Automatic endpoint tagging
- Metadata enrichment (base URL, auth)
- API template generation
- Minimal UI for testing

## How It Works

OpenAPI JSON → Parse → Normalize → Tag → Enrich → Generate Templates
